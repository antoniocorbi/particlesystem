# Particle System Simulator

Just that!
A Particle System Simulator.

## Compile and run:

``` shell
cargo run
```

    - Adjust the parameters and click to create a new Particle System or
      click & drag to create one and interactively add new Particles.

    - The particles get darker as their age increases. When they are
      `black` the particle dies and is removed from de simulation.

    - Also when a Particle System has 0 Particles in it it is also
      removed (you can check bopth counters -particles and particlesystem-).
